Operation Guidance

  1. Prerequisites

    • A Pixel 3 emulator with API 23 with target app installed
    • OpenAI API Key
    • Python 3.8

  2. Usage

    • Install the required packages: 'pip install -r requirements.txt'.
    • Populate the 'code/config.py' file with the information of device name and gpt api key.
    • Set the objects to be migrated for this ContextMig run in the 'subjects' variable of 'migrate_subjects.py', with the format: 'source app_test_target app'.
    • Run the 'code/main.py' file.
    • You can view the running logs of the six agents (test semantic analyzer, event contextual semantic analyzer, event selector/validator, test script generator/validator) in the console or in the 'log/run.log' file.
    • After the migration is complete, you will find a markdown file summarizing the workflow of this migration in the folder 'dataset/result/'.

Running Example

This example illustrates the migration of a test case between two shopping list management applications, demonstrating the migratability of a test case for sorting items in descending order by price from a source application to a target application.
In the source app, the test case entails a sequence of operations: navigating to 'Family', proceeding to 'Menu', then 'Settings', selecting 'Sort order', choosing 'Most expensive first', and finally clicking 'Back'.
In the target app, the corresponding sequence after migration involves navigating to 'Family', then 'Menu', selecting 'Sort', choosing 'Descending', specifying 'Price', and clicking 'OKAY'.

Here is the detailed workflow process for migrating the running example using ContextMig:
  • The 'Semantics of Source Test Case' is extracted by the Test Semantic Analyzer.
  • The 'Contextual Semantics of Widgets' is obtained through the Event Contextual Analyzer.
  • The 'Selected Event(s) and Script' is derived using the Event Selector and the Test Script Generator.
  • The 'Migration Success' denotes that the migration process is successful and includes the scripts that are executed during the process.
